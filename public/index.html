About Me

Abstract

Fullstack developer from Brazil, 18 yo. I started with C++ because I wanted to learn the hardest first, then I gone to PHP, I learned Javascript for web and later I replaced PHP by node, then I learned Typescript and I used it to almost all, but at some time ago, I found the Nim, and now I use it mainly for frontend, backend and IoT.

I'm around 4 years in this area, and I like very much, I think that it's one of the best skills to have in this age

Motivation

I really like programming, I always thought how the games works, how complex it is? Then I decided to learn programming, not to make games, just to see how it was made.

Then I made some games, but the real interest is have that "ability"

Later I started to make things to my dad and I learned more deep: Web and IoT


It hasn't been that long since I start to contribute at open-source community, recently I published a lot of projects who where on my machine.
